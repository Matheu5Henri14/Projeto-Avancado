#  Projeto Final: HTML e CSS para Iniciantes

Este projeto é o resultado final do curso de HTML e CSS da **Origamid**. Ele marca a conclusão da minha base fundamental em front-end, onde apliquei conceitos avançados de estruturação e design responsivo para criar uma interface moderna e funcional.

---
##  Acessando Através do GitHub Pages
* Acesse por esse link: (https://matheu5henri14.github.io/Projeto-Avancado/)
---

##  Objetivos Concluídos
* **Semântica HTML5**: Estrutura de código limpa e otimizada para SEO e acessibilidade.
* **Layouts com Flexbox e Grid**: Domínio total de posicionamento e alinhamento de elementos.
* **Responsividade**: Site totalmente adaptado para dispositivos móveis, tablets e desktops.
* **Tipografia e Cores**: Aplicação de variáveis CSS e boas práticas de design visual.

---

##  Tecnologias Utilizadas
* **HTML5**: Estruturação de conteúdo.
* **CSS3**: Estilização, animações básicas e layouts.
* **Figma**: Utilizado para seguir o protótipo e garantir fidelidade visual.

1. ⏳ **JavaScript Completo**: Adição de interatividade e comportamento.
